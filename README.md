# Excel Salary Dashboard

## Introduction
An interactive Excel dashboard analyzing data science salaries across job roles, countries, and employment types.

## Skills Used
- Excel Charts
- Pivot Tables
- Data Validation
- Formulas & Functions
- Dashboard Design

## Dataset
The dataset contains data science job salary information including:
- Job Titles
- Salaries
- Countries
- Skills

## Key Insights
- Senior-level roles showed higher median salaries.
- Salary trends varied significantly across countries.
- Different employment types influenced salary distribution.
